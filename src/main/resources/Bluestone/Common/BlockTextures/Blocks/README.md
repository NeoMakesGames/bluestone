# Block Textures

Place the following texture files in this directory:

## Bluestone Wire Textures
- `bluestone_wire_off.png` - Texture for unpowered wire (dark/dull blue)
- `bluestone_wire_on.png` - Texture for powered wire (bright blue, possibly glowing)

## Bluestone Switch Textures
- `bluestone_switch_off.png` - Texture for switch in OFF position
- `bluestone_switch_on.png` - Texture for switch in ON position

## Texture Specifications
- Format: PNG
- Recommended size: 16x16 or 32x32 pixels
- Use transparency where needed (especially for wire cross pattern)
